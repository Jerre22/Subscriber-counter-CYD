# YouTube Subscriber Counter

A desktop YouTube subscriber counter built with:

- ESP32
- ST7789 2.8" Display
- YouTube Data API v3

## Features

- Live subscriber count
- Automatic milestone tracking
- Progress bar
- WiFi enabled

## Hardware

- ESP32-32E
- ST7789 Display
- USB-C Power
